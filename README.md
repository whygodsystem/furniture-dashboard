# Финансовый дашборд мебельной компании

Интерактивный дашборд для анализа финансовых показателей производства мебели.

## Установка и запуск

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Сгенерируйте тестовые данные: `python generate_data.py`
4. Запустите приложение: `python app.py`
5. Откройте `http://127.0.0.1:8050/` и загрузите файл `furniture_data.csv`

## Элементы дашборда

- Загрузка данных через интерфейс
- 4 KPI-карточки (выручка, расходы, прибыль, заказы)
- Линейный график динамики доходов и расходов
- Круговая диаграмма структуры расходов
- Гистограмма распределения прибыли
- График рассеяния (корреляция выручки и прибыли)
- Детальная таблица по заказам
- Фильтры по периоду (месяц/квартал/год) и категориям
